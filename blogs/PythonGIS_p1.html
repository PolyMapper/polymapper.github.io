Utilising GIS functions within Python - Part 1 – Creating and using basic geometry

January 12, 2024 by Gordon McLachlan

The aim of the next series of posts are to give an understanding of the utilising GIS functions within Python to manipulate, analyse, and ultimately present spatial data. In this series I will go from discussing basic vector geometry, through to raster analysis, and on to creating static and interactive web map documents. The first part will illustrate how to create and use basic geometry.

-->

Points

The basic elements of vector geometry are points, lines, and polygons. In this post I will briefly demonstrate how to create and interact with this type data.

Firstly, it is necessary to import the relevant geometries from the Shapely module:

										# Import necessary geometric objects from shapely module 
										from shapely.geometry import Point, LineString, Polygon

To create point geometries we use the Point object.

										# Create point geometries based on inputted coordinates
										point1 = Point(2.2, 4.2)

										point2 = Point(7.2, -25.1)

										point3 = Point(9.26, -2.456)

To see the point, you can print:

										# view the inputted data
										print (point1)
										[out]: POINT (2.2 4.2)

You can also view the type of data. This is important check as you, as the user, may not be sure what the data is. So, whenever you import new data, it is always worth running this query:

										# you can query the type of data
										point_type = type(point1)

It is also possible to create points with a 3rd dimension:

										# It is also possible to add a 3rd dimension to point data
										point3D = Point(9.26, -2.456)

The Point object comes with a number of abilities and functions. The most important of these are to extract coordinates and then calculate the Euclidean distance between two sets of coordinates.

The first step is to extract the points from the coordinates:

										# extracting the coordinates
										point_coords = point1.coords 
										
										# Checking the datatype
										print(type(point_coords))
										[out]: shapely.coords.CoordinateSequence

It is also possible to extract the X and Y values.

										# get the x and y coordinates
										xy = point_coords.xy
										
										# get only the x coordinate 
										x = point1.x 
										
										# get only the y coordinate 
										y = point1.y

Here, xy is a tuple data type, while X and Y are plain decimal numbers.

We can also calculate the distance:

										# to calculate the euclidean distance between points 
										point_dist = point1.distance(point2) 
										
										# We can make this a bit more readable 
										print("Distance between the points is {} decimal degrees".format(point_dist)) 
										[out]: Distance between the points is 29.723559679150142 decimal degrees 
										
										# remove all the decimal places (to two points) 
										print("Distance between the points is {0:.2f} decimal degrees".format(point_dist)) 
										[out]: Distance between the points is 29.72 decimal degrees

LineString

Creating LineString objects are done in a similar way to how Point objects are created, except of using a single Tuple, we use multiple points or tuples.

										#Create a LineString from the Point objects instantiated above 
										# we use the square brackets as they represent a list 
										# while parentheses are tuples (you cannot append to amend tuples) 
										line = LineString([point1, point2, point3]) 
										
										# it is also possible to input tuple coordinates into the list directly 
										line2 = LineString([(2.2, 4.2), (7.2, -25.1), (9.26, -2.456)])

We can look at the lines using the following:

										# what do these look like: 
										print(line) 
										[out]: LINESTRING (2.2 4.2, 7.2 -25.1, 9.26 -2.456) 
										
										print(type(line)) 
										[out]: shapely.geometry.linestring.LineString 
										
										print(line2) 
										[out]: LINESTRING (2.2 4.2, 7.2 -25.1, 9.26 -2.456)

We can extract the coordinates of the line

										# Extract the x and y coordinates of the line 
										lxy = line.xy print(lxy) 
										[out]: (array('d', [2.2, 7.2, 9.26]), array('d', [4.2, -25.1, -2.456]))

We can also get particular points by selecting a value from an array.

										line_x = lxy[0] 
										
										# Extract y coordinates straight from the LineObject by referring to a array at index 1 
										line_y = line.xy[1] print(line_x) 
										[out]: array('d', [2.2, 7.2, 9.26]) 
										
										print(line_y) 
										[out]: array('d', [4.2, -25.1, -2.456])

You can also calculate the length of lines

										# Get the lenght of the line 
										l_length = line.length 
										
										# Get the centroid of the line 
										l_centroid = line.centroid 
										
										# What type is the centroid? 
										centroid_type = type(l_centroid) 
										
										# Print the outputs 
										print("Length of our line: {0:.2f}".format(l_length)) 
										[out]: Length of our line: 52.46 
										
										print("Centroid of our line: ", l_centroid) 
										[out]: Centroid of our line:  POINT (6.229961354035622 -11.89241115757239)

Polygon

Polygons use the same logic, however, it only accepts coordinate inputs as tuples, and it needs at least three inputs.

										#Create a polygon based on inputted coordinates 
										poly = Polygon([(2.2, 4.2), (7.2, -25.1), (9.26, -2.456)]) 
										
										# We can also use our previously created Point objects 
										# it should be noted that the Polygon object requires x,y coordinates as inputs 
										# this looks through the x and ys in the points 
										poly2 = Polygon([[p.x, p.y] for p in [point1, point2, point3]]) 
										
										#Geometry types can be accessed and outputted as String values 
										poly_type = poly.geom_type 
										
										#Using the Python type function, we are able to get the same information in a different format 
										poly_type2 = type(poly) 
										print("Geometry type as text:", poly_type) 
										[out]: Geometry type as text: Polygon 
										
										print("Geometry how Python shows it:", poly_type2) 
										
										# Lets look at the polygon 
										print(poly) 
										[out]: POLYGON ((2.2 4.2, 7.2 -25.1, 9.26 -2.456, 2.2 4.2)) 
										
										print(poly2) 
										[out]: POLYGON ((2.2 4.2, 7.2 -25.1, 9.26 -2.456, 2.2 4.2))

We can also create polygons with interior holes

										# The above polygon contains no holes 
										# we can create polygon with interior holes 
										# defining the exterior 
										world_exterior = [(-180, 90), (-180, -90), (180, -90), (180, 90)] 
										
										# now create the interior 
										hole = [[(-170, 80), (-170, -80), (170, -80), (170, 80)]] 
										
										# world polygon without a hole 
										world = Polygon(shell=world_exterior) 
										
										# Now we can construct our polygon with a hole inside 
										world_has_a_hole = Polygon(shell=world_exterior, holes=hole) 
										
										# to visualise what we have 
										print(world) 
										[out]: POLYGON ((-180 90, -180 -90, 180 -90, 180 90, -180 90)) 
										
										print(world_has_a_hole) 
										[out]: POLYGON ((-180 90, -180 -90, 180 -90, 180 90, -180 90), (-170 80, -170 -80, 170 -80, 170 80, -170 80)) 
										
										print(type(world_has_a_hole)) 
										[out]: shapely.geometry.polygon.Polygon

We can do a number of different queries on these polygons to gain more information about them

										# get the centroid of the polygon 
										world_centroid = world.centroid 
										
										# we can now calculate the area 
										world_area = world.area 
										
										# extracting the bounding box of the polygon 
										world_bbox = world.bounds 
										
										# get the exterior of the polygon 
										world_ext = world.exterior 
										
										# get the perimeter length of the exterior 
										world_ext_length  = world_ext.length 
										
										# now to see what we have 
										print("Poly centroid: ", world_centroid) 
										[out]: Poly centroid:  POINT (-0 -0) 
										
										print("Poly Area: ", world_area) 
										[out]: Poly Area:  64800.0 
										
										print("Poly Bounding Box: ", world_bbox) 
										[out]: Poly Bounding Box:  (-180.0, -90.0, 180.0, 90.0) 
										
										print("Poly Exterior: ", world_ext) 
										[out]: Poly Exterior:  LINEARRING (-180 90, -180 -90, 180 -90, 180 90, -180 90) 
										
										print("Poly Exterior Length: ", world_ext_length) 
										[out]: Poly Exterior Length:  1080.0
